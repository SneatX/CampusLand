Tipos de datos

  1. number : Para almacenar numeros
  2. undefined : Cuando no se le asigna un valor a una variable
  3. null : Cuando un dato no existe
  4. string : Representa texto (con comillas simples, dobles o acentos invertidos)
  5. boolean : tipo true o false

Para crear variables se usa "let"
